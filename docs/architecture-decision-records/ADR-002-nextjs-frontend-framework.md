# ADR-002: Use Next.js as the Frontend Framework

**Status:** Accepted

## Context

The platform requires excellent SEO, performance, scalability, and long-term community support.

## Options Considered

* Next.js
* Astro
* Remix
* Nuxt

## Decision

Use Next.js with the App Router.

## Rationale

Next.js provides server components, static generation, image optimisation, strong SEO capabilities, and a mature ecosystem that aligns with the project's long-term goals.

## Consequences

The platform benefits from modern rendering strategies while remaining familiar to contributors experienced with React.
