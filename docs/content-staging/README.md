# Content Staging

Working notes for real-world content that has been provided but not yet entered into Sanity, because it belongs to a Phase 3 step (see `docs/10-roadmap.md` and `README.md`) that hasn't been reached yet.

Each file holds raw notes, organised by the entity type they'll eventually become. Nothing here is published content — it exists so information given during one phase isn't lost before its phase arrives. Once a note's content has been entered into Sanity, remove it from the file (or delete the file if nothing remains).

None of this content should be treated as verified. Verification Status is assigned only when the content is actually entered, per the Editorial Policy.
